Инструкция для пользователя.

1) Введите границы интегрирования.
2) Введите желаемый шаг вычислений.
3) Выберите метод(-ы) для решения дифференциальных уравнений. (Необходимо отметить галочкой нужный(-е) пункт(-ы))
1. Метод Эйлера
2. Метод Рунге-Кутты второго порядка
3. Метод Рунге-Кутты четвёртого порядка
4. Метод Рунге-Кутты-Фельберга
4) Выберит или готовую функцию, или нажмите на галочку "Пользовательская функция" и напишите свою в поле для ввода.
5) Нажмите кнопку "Рассчитать".
Требования:
Библиотека libxl
Windows x64

Во вкладке "Настройки" вы сейчас находитесь.
Во вкладке "График" у вас появится график по всем выбранным методам.
Во вкладке "Таблица" у вас повявятся заполненные таблицы вычислений пошагово каждого из методов.
Во вкладке "Импорт" у вас есть возможность вывести данные в exel файл, в нём на каждом листе будут результаты решений по каждому методу соответственно.
